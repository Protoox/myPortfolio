My portfolio
My first project is an exciting personal portfolio that I have developed using a combination of cutting-edge technologies. On the frontend, I have utilized Angular. As for the backend, I have chosen to use Spring. With Spring, I have built a customized API that serves as an intermediary between the frontend and a MySQL database. This API handles incoming requests, performs the necessary database queries, and sends the corresponding responses back to the frontend. I leverage this API to extract data related to my skills, experience, and other relevant information for showcasing in my portfolio. Check the API
